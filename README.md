# ACNH-Discord-Bot

Discord Bot that returns infortmation on items (ex. fish, bugs, art etc.) found in the ACNH database.

Example of How to Use:
```
!fish dace
Name: DACE
Price: 240 Bells 
Location: River 
Rarity: Common 
```

Built using Python.
